#bbmanage
个人后台管理项目，并为微信小程序项目提供接口API

###前端使用技术
* 后台页面
 * 感谢H-ui 提供静态页面支持
 * Ztree：jQuery树插件
 * DataTables：jQuery表哥插件
 * FullCalendar：日期插件
###后段所用技术
* Spring boot
* Spring Data Jpa
* Spring Security
* MySQL
* Redis
* Druid 阿里高性能数据库连接池
* Spring 
* Maven 
* 第三方SDK（基本上都是使用免费的，个人开发，学习使用）
    * 七牛云存储服务
    * 极验Test-button人机验证码 
    * 网易云短信
